02 // Menggunakan Tag Anchor <a>

Untuk membuat tautan, HTML menggunakan tag <a> (Anchor) yang dikombinasikan dengan atribut wajib href.

<a href="https://courslan.com"> Kunjungi Website </a>

Tag <a>

Membungkus teks atau elemen visual lain agar bisa diklik oleh user.

Atribut href

Singkatan dari *Hypertext Reference*. Menyimpan alamat tujuan utama link tersebut.

Anchor Text

Teks di antara tag pembuka & penutup yang tampil secara visual sebagai link berwarna biru/garis bawah.

03 // Absolute Path vs Relative Path

Menentukan nilai href membutuhkan pemahaman struktur direktori komputer atau server internet.

ABSOLUTE PATH (ALAMAT LENGKAP)

Digunakan untuk menunjuk ke website eksternal di luar server kita. Wajib menyertakan protokol HTTP/HTTPS.

<a href="https://google.com">Google</a>
RELATIVE PATH (ALAMAT LOKAL)

Menunjuk ke file internal di project folder kita sendiri. Tidak butuh nama domain.

<a href="about.html">Tentang Kami</a>

04 // Atribut target="_blank"

Secara default, saat link diklik, halaman baru akan terbuka di tab yang sama dan menumpuk web lama Anda.

Gunakan atribut target="_blank" agar browser membuka tautan tersebut di Tab Baru, sehingga pengunjung tidak meninggalkan website utama Anda.

HTML IMPLEMENTATION
<a
  href="https://github.com"
  target="_blank">
  Buka Kode Repository
</a>

05 // Menyematkan Gambar <img>

Gambar membuat website lebih hidup. Untuk menampilkannya kita menggunakan tag <img>. Tag ini bersifat Self-Closing (Tidak butuh tag penutup </img>).

<img src="logo.png" alt="Logo Akademi Courslan" width="300">
Atribut src (Source): Menyimpan lokasi path/URL gambar berada.
Atribut alt (Alternative Text): Teks pengganti jika gambar gagal dimuat, sekaligus dibaca oleh bot Google SEO dan Screen Reader disabilitas.

06 // Menyematkan Audio <audio>

HTML5 mempermudah kita memutar file suara tanpa plugin flash tambahan dengan menggunakan tag <audio>.

Wajib menyertakan atribut controls agar browser memunculkan bar tombol Play, Pause, dan Volume Mute.

AUDIO STRUCTURE
<audio controls>
  <source src="podcast.mp3" type="audio/mpeg">
  Browser Anda tidak mendukung audio.
</audio>

07 // Menyematkan Video <video>

Mirip dengan audio, tag <video> digunakan untuk memutar video lokal (seperti .mp4 atau .webm) langsung di dalam halaman website.

<video width="640" height="360" controls poster="thumbnail.jpg">
  <source src="tutorial.mp4" type="video/mp4">
</video>

Atribut poster berfungsi untuk menampilkan gambar thumbnail sebelum video dimainkan oleh user.
