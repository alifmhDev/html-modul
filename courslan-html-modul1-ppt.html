02 // Anatomi Sebuah Tag

<p class="info"> Halo Courslan </p>

1. Tag Pembuka

Menandai dimulainya sebuah elemen. Ditulis dengan kurung siku < >.

2. Konten

Informasi aktual (seperti teks atau gambar) yang dibungkus dan akan tampil di layar pengguna.

3. Tag Penutup

Wajib memiliki garis miring / sebelum nama tag untuk menutup area elemen tersebut.

03 // Atribut: Metadata Tambahan

Atribut memberikan instruksi ekstra pada elemen. Selalu diletakkan di dalam Tag Pembuka dan formatnya berpasangan nama="nilai".

<a href="https://courslan.com" target="_blank"> Menuju Academy </a>

Nama Atribut:
Menentukan *apa* yang diatur (misal: href untuk link, src untuk gambar).
Nilai Atribut (Value):
Spesifikasi pengaturannya. Selalu dibungkus tanda kutip ganda "...".

04 // Struktur Wajib (Boilerplate)

<!-- Deklarasi Versi -->
<!DOCTYPE html>
<html lang="id">
  <head>
    <title>Web Saya</title>
  </head>
  <body>
    Konten Anda di sini...
  </body>
</html>
  • !DOCTYPE: Perintah untuk browser agar merender ini sebagai HTML5 terbaru.
  • <html>: Tag wadah akar (root) untuk seluruh elemen.
  • <head>: Otak dokumen. Berisi metadata, pemanggilan file CSS, dan judul Tab. Area ini tidak terlihat oleh user.
  • <body>: Tubuh web. Semua teks, video, & tombol yang terlihat di layar harus masuk ke dalam sini.

05 // Tipografi: Heading (H1-H6)

Heading mendefinisikan *hierarki informasi*, mirip seperti bab dalam sebuah buku. Ingat, heading digunakan untuk struktur, bukan untuk sekadar memperbesar ukuran font!

<h1> Judul Utama Halaman </h1>
<h2> Sub-Judul Utama </h2>
<h3> Sub-bagian dari H2 </h3>
SEO RULE: Gunakan hanya SATU H1 per halaman. Ini membantu mesin pencari Google (SEO) untuk langsung memahami apa topik utama dari halaman website Anda.

07 // Whitespace & Indentasi

Browser akan mengabaikan spasi berlebih atau baris baru kosong di dalam kodemu. Lalu mengapa kita perlu merapikannya? Demi manusia (Rekan Tim Developer).

BAD CODE (SUSAH DIBACA)

<body><h1>Judul</h1><p>Teks</p><div><p>Subteks</p></div></body>

GOOD CODE (CLEAN)

<body>
  <h1>Judul</h1>
  <p>Teks</p>
  <div>
    <p>Subteks</p>
  </div>
</body>

Gunakan 2 atau 4 spasi (tombol Tab) untuk elemen yang berada di dalam elemen lain (*Nesting*).
