01 // Gejala "Div Soup"

Dulu, developer merancang layout website dengan pakai <div> berulang-ulang untuk segala hal. (Disebut Div Soup).

Bagi Google Bot, <div> tidak memiliki arti apa-apa selain kotak kosong. Mesin tidak tahu mana area menu atau konten utama.

02 // Apa itu Semantic HTML?

Semantic berarti "makna". Tag Semantik adalah tag HTML yang namanya langsung menjelaskan jenis konten di dalamnya kepada browser.

Non-Semantic (Tanpa Arti)

<div> dan <span>

Hanya memberi tahu browser "Ini adalah sebuah wadah".

Semantic (Punya Makna)

<form>, <table>, <article>

Menjelaskan "Ini adalah area formulir, tabel, dan artikel berita."

03 // Elemen Web Modern

Gantikan `<div id="header">` dengan Tag Semantik bawaan HTML5 berikut untuk kerangka website Anda:

<header>

Bagian atas halaman web. Berisi Logo atau form pencarian.

<nav>

Wadah khusus untuk deret menu link navigasi utama.

<footer>

Catatan kaki halaman. Berisi hak cipta & link sosial media.

04 // Pembungkus Konten Inti

<main>

Konten inti paling penting. Hanya boleh ada SATU per halaman.

<section>

Memecah `main` ke beberapa bagian (Section Layanan/Portfolio).

<article>

Konten mandiri utuh (Misal: Postingan Blog).

MODERN SKELETON
<body>
  <header><nav>...</nav></header>
  <main>
    <section>
      <article> Post 1 </article>
    </section>
  </main>
  <footer> Copy 2026 </footer>
</body>

05 // Mengapa Semantic Penting?

1. SEO (Search Engine Optimization)

Google Bot mendeteksi tag `<main>` dan `<article>` untuk menentukan konten prioritas mana yang akan dinaikkan rankingnya ke Halaman Pertama Google.

2. Aksesibilitas (A11Y)

Software pembaca layar tunanetra akan mendeteksi `<nav>` dan membacakannya, sehingga penyandang disabilitas bisa menggunakan web Anda.

06 // Tag <head>: Otak Tersembunyi

Konfigurasi krusial website tidak terjadi di area visual (body), melainkan di dalam tag <head>.

Area ini menampung Metadata ("data tentang data"). Ia memberitahu browser cara memuat halaman serta memberi info ke sosmed saat link web dibagikan.

METADATA_INSPECTION
  • Tidak merender elemen visual.
  • Memuat font eksternal (Google Fonts).
  • Menautkan file gaya/desain (CSS).

07 // Meta Tags Wajib SEO

Di dalam <head>, kita menanamkan tag tunggal <meta>. Berikut adalah 3 Meta Tag wajib web modern:

<!-- 1. Mencegah karakter teks rusak -->
<meta charset="UTF-8">

<!-- 2. Wajib untuk Mobile Responsif -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 3. Teks deskripsi untuk pencarian Google -->
<meta name="description" content="Situs belajar Courslan.">
