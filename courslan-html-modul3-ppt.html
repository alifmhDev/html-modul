04 // Anatomi Dasar Tabel

Membuat matriks data dua dimensi membutuhkan kombinasi tiga tag inti HTML: <table>, <tr>, dan <td>.

<table>
  <tr>
    <td>Arslan</td>
    <td>Web Dev</td>
  </tr>
</table>
<table> : Pembungkus utama seluruh area matriks tabel.
<tr> : Table Row. Berfungsi menciptakan satu baris horizontal baru.
<td> : Table Data. Berfungsi membuat sel data kolom vertikal di dalam baris.

05 // Menambahkan Header <th>

Sebuah tabel profesional wajib memiliki kepala kolom yang jelas agar data mudah dibaca. Kita mengganti tag <td> pada baris pertama dengan tag <th> (Table Header).

<table>
  <tr>
    <th>Nama Kelas</th>     <th>Tipe Akses</th>
  </tr>
  <tr>
    <td>HTML Fundamental</td>  <td>Gratis (Rutinan)</td>
  </tr>
</table>

Kelebihan <th>: Browser secara otomatis akan membuat teks di dalam elemen ini menjadi bercetak tebal (Bold) dan diposisikan tepat di tengah sel (Centered) secara default tanpa CSS.

07 // Menggabungkan Sel: Spanning

Sama seperti fitur *Merge Cells* pada Microsoft Excel, HTML memiliki dua atribut andalan untuk menggabungkan beberapa sel menjadi satu kesatuan.

colspan (Mendatar / Horisontal)

Menggabungkan beberapa kolom menjadi satu baris sel besar.

<td colspan="2">Gabung 2 Kolom</td>
rowspan (Tegak / Vertikal)

Menggabungkan beberapa baris ke bawah menjadi satu kolom sel tinggi.

<td rowspan="2">Gabung 2 Baris</td>

*Peringatan:* Saat menggunakan spanning, Anda harus menghapus tag <td> tetangga yang tergeser agar struktur tabel tidak rusak mencuat keluar.
